# FD Design System Fonts

This directory should contain the following font files from the FD Design System:

## Required Font Files:

### Proxima Nova (Sans-serif)
- `ProximaNovaRegular.ttf` or `ProximaNova-Regular.ttf`
- `ProximaNovaBold.ttf` or `ProximaNova-Bold.ttf`

### Arnhem Pro (Serif)
- `ArnhemProBlond.ttf` or `ArnhemPro-Blond.ttf` (Regular serif)
- `ArnhemProBold.ttf` or `ArnhemPro-Bold.ttf` (Bold serif)

## How to Obtain Fonts:

1. Contact FD Design System team or check their design system documentation
2. Download from FD's internal font repository
3. Ensure you have proper licensing for these fonts

## Font File Naming:

The fonts are configured in `pubspec.yaml` to work with common naming conventions. If your font files have different names, you may need to update the `pubspec.yaml` configuration.

## Note:

These fonts are proprietary and require proper licensing. Make sure you have the rights to use them in your application.
